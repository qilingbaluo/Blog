# Django博客练习

基于django实现的博客
* 编程语言  
    * 脚本语言  
        * Python 
    * 前端语言
        * html
        * css
        * JavaScript

版本配置：
    Django2.x
    python3.7

功能介绍：
- 登录注册
- 文章管理
- 个人信息管理
- 文章查看

运行代码：
安装相同配置的django和python
迁移数据库
```python
python manage.py makemigrations
python manage.py migrate
```
运行项目
```python
python manage.py runserver
```
运行成功后会显示路由：127.0.0.1:8000
